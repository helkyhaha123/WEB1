WEB1

WEB1 adalah aplikasi web sederhana yang dibuat menggunakan PHP dan MySQL. Aplikasi ini menyediakan fitur autentikasi pengguna serta manajemen data melalui antarmuka yang mudah digunakan.

Fitur

- Login pengguna
- Registrasi akun
- Dashboard pengguna
- Tambah data
- Edit data
- Hapus data
- Upload file/gambar
- Logout

Teknologi yang Digunakan

- PHP
- MySQL
- HTML
- CSS
- XAMPP / Apache

Instalasi

1. Clone repository ini.
2. Import file "database.sql" ke MySQL.
3. Sesuaikan konfigurasi database pada file "config.php".
4. Jalankan Apache dan MySQL.
5. Akses aplikasi melalui browser.

Struktur Project

- "index.php" : Halaman utama
- "login.php" : Halaman login
- "register.php" : Halaman registrasi
- "dashboard.php" : Dashboard pengguna
- "tambah.php" : Menambah data
- "edit.php" : Mengubah data
- "hapus.php" : Menghapus data
- "upload/" : Penyimpanan file upload
- "config.php" : Konfigurasi database
